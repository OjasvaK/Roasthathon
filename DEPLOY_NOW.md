## 💡 Pro Tips

**MongoDB Connection String**
- Format: `mongodb+srv://user:password@cluster/db?options`
- Save it safely - you'll need it for Vercel
- Passwords with special chars may need URL encoding

**Environment Variables**
- Set in Vercel dashboard BEFORE deployment
- Redeploy automatically after setting
- Check "Environment Variables" tab in Vercel

**First Deployment Takes Longer**
- Initial build: 2-3 minutes
- Later deployments: 1-2 minutes
- Watch the Vercel build log
