# Roasthathon

- Retro 8-bit themed project showcase and idea forum.
- Built as a static frontend experience with no backend.
- Includes login, project listings, idea proposals, and profile view.

## Features

- Login screen with hardcoded demo credentials.
- Projects tab showing project cards and author filtering.
- Ideas tab for proposal browsing and category filtering.
- Profile tab with user identity and stats display.
- Pixel-art interface with glassmorphism styling.

## Files

- `styles.css` - visual theme, layout, and animations.

## Usage

1. Open the site using its deployed URL.
2. Enter the demo credentials on the login screen.
3. Explore projects, ideas, and profile sections.

## Notes

- The app is static and intended for presentation/demo use.
- Login is for UI flow only; no server-side authentication is included.
